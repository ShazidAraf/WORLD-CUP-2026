How the logic works: win = 3 pts, draw = 1. Ties are broken by head-to-head, then seeding (a click tool has no goal difference, which is FIFA's first real tiebreaker). The best-8 thirds are ranked by points, then group order — adjustable by tapping. The third-place Round-of-32 slots are auto-arranged into a valid configuration from your eight qualifiers (FIFA uses a fixed 495-row lookup table for the official allocation).

Probability bars: each match shows model win / draw / loss odds from current World Football Elo ratings (worldfootballrankings.com, as of 11 June 2026) — Elo updates after every match, so it reflects recent form and squad strength. Odds use an independent-Poisson goals model anchored to the Elo expected score (~2.7 goals/game), with a modest home boost for the three host nations. In the knockouts a draw goes to extra time / penalties, so each side's “advance” figure splits the draw evenly. A * marks ten lower-ranked qualifiers whose current rating is approximate. Estimates for fun — not betting advice. Two auto-fill buttons use these odds: 📈 Most likely always takes the single most probable result (the favourite advances; no draws), giving the same “chalk” bracket every time; 🎯 Auto-pick by odds samples each result from its probabilities, so favourites usually advance but draws and upsets appear in proportion (a different bracket each run).
